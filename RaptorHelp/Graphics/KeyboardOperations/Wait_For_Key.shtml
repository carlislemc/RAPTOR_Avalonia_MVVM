Wait_for_Key

 

Wait_For_Key is a call that pauses the program until a key has been pressed in the RaptorGraph window.  

 

Notes:  A graphics window must be open when Wait_For_Key is called or a run-time error will occur.  If the RaptorGraph window is not on top, the user may have to click on it once before it will respond to user keystrokes.

 

Example:

 

Wait_For_Key
