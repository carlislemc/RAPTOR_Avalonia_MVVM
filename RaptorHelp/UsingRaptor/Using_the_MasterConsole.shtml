Using the MasterConsole

The MasterConsole window displays all user input and output.  Input and output is achieved using the input or output symbol.

 

 

The small textbox at the bottom of the MasterConsole window allows the user to directly input statements and commands during execution.  For example, the user may want to see what a RaptorGraph operation will do by typing in a procedure call in this textbox (e.g. "Open_Graph_Window(300,300)").

 

The Clear button clears the MasterConsole window.
